# FinTrack Backend

API para controle financeiro pessoal, permitindo o registro de ganhos,
gastos e visualização de saldo.

## Objetivo
Construir um backend simples e escalável, com foco em boas práticas,
qualidade de código e testes de API desde o início.

## Tecnologias
- Node.js
- Express
- SQLite
- JWT (autenticação)

## Funcionalidades (MVP)
- Cadastro e login de usuário
- Registro de ganhos
- Registro de gastos
- Listagem de transações
- Cálculo de saldo
